API

To upload using curl or make a tool you can post using:

  • curl -i -F files=@yourfile.jpeg https://0x80.eu.org/upload.php (JSON Response)

  • curl -i -F files=@yourfile.jpeg https://0x80.eu.org/upload.php?output=text (Text Response)

  • curl -i -F files=@yourfile.jpeg https://0x80.eu.org/upload.php?output=html (HTML Response)

Example response API:

            {
             "success" : true,
             "name" : "yourfile.jpeg",
             "files" : [
              "https://a.0x80.eu.org/W93jHr.jpeg",
              "https://ipfs.io/ipfs/bafkreie2xbsrwpz2pmciax2t6gdk3omnlhlklu..."
             ]
            }
